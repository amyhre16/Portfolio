Nomz

Nomz is a data-persistant, nutritional shopping list. Nomz provides data unique to the user. When an item is added to the list, its nutritional information is displayed in a table. Using the user's location, a map is displayed showing the stores closest to the user. Trello was used to manage project tasks.
Collaborated with a group of 3 other developers.

Technologies Used

  • HTML5
  • CSS3
  • jQuery
  • Bootstrap
  • Express
  • Google Firebase
  • Edamam API
  • Google Maps API
  • Request NPM Package

Train Schedule

This app provides the user to keep track of their train schedule. When a train is entered into the system, the app calculates the when the next train will arrive and how many minutes are left until it arrives.
Worked independently.

Technologies Used

  • HTML5
  • CSS3
  • jQuery
  • Bootstrap
  • Moment.js
  • Google Firebase

Hot Restaurant

This restaurant waiting list app was built in 3 hours during class. A database is used to store reservations. Once there are 5 reservations, the table is added to a waiting list. The user can view the reservation/waiting list and clear both lists at any time.
Collaborated with 2 other developers.

Technologies Used

  • HTML5
  • CSS3
  • jQuery
  • Bootstrap
  • Express
  • MySQL
  • ORM Database Model
  • JawsDB
